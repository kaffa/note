## 琅環笔记

### 介绍

琅環笔记是我的个人笔记，这个目录包含琅環笔记网站的源码。

笔记主要内容是一人开发模式中摸过的石头和踩过的坑，英文可以叫做 Tips 或 Traps，或 The Missing Manual，或 Undocumented Docs。


### 关于 GitHub 发布

Nikola GitHub 发布的推荐做法是分为 main 分支和 src 分支：

1. src 分支是「马戏团后台」
2. main 分支来自 src 分支的 output 目录

GitHub Pages 每次发布 main 分支。

关于这种做法的评价，我放在：https://note.tt4e.com/posts/nikola-tips/#nikola-github
